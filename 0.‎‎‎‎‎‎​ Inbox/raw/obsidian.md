Perfeito! No **Obsidian**, você pode usar várias formatações de texto com **Markdown**. Aqui vai um guia organizado com exemplos:

---

### 1️⃣ Negrito

```markdown
**Texto em negrito**
__Texto em negrito__
```

Exemplo: **Importante** ou **Importante**

---

### 2️⃣ Itálico

```markdown
*Texto em itálico*
_Texto em itálico_
```

Exemplo: *Dica* ou *Dica*

---

### 3️⃣ Negrito + Itálico

```markdown
***Texto em negrito e itálico***
___Outro exemplo___
```

Exemplo: ***Urgente*** ou ***Urgente***

---

### 4️⃣ Tachado (strikethrough)

```markdown
~~Texto riscado~~
```

Exemplo: ~~Errado~~

---

### 5️⃣ Sublinhado (via HTML)

```markdown
<u>Texto sublinhado</u>
```

Exemplo: <u>Sublinhado</u>

*(Markdown puro não tem sublinhado, mas HTML funciona no Obsidian)*

---

### 6️⃣ Código inline

```markdown
`código`
```

Exemplo: `console.log("Olá")`

---

### 7️⃣ Bloco de código

````markdown
```javascript
function ola() {
  console.log("Olá mundo");
}
````

````
Exemplo:  
```javascript
function ola() {
  console.log("Olá mundo");
}
````

---

### 8️⃣ Citações

```markdown
> Esta é uma citação
```

Exemplo:

> Esta é uma citação

---

### 9️⃣ Listas

* **Lista não ordenada:**

```markdown
- Item 1
- Item 2
```

Exemplo:

* Item 1

* Item 2

* **Lista ordenada:**

```markdown
1. Primeiro
2. Segundo
```

Exemplo:

1. Primeiro
2. Segundo

---

### 1️⃣0️⃣ Links e imagens

* **Link:**

```markdown
[Google](https://www.google.com)
```

Exemplo: [Google](https://www.google.com)

* **Imagem:**

```markdown
![Alt text](https://link-da-imagem.com/img.png)
```

Exemplo:
![Exemplo](https://via.placeholder.com/150)

---

### 1️⃣1️⃣ Tabelas

```markdown
| Nome  | Idade |
|-------|-------|
| Ana   | 25    |
| João  | 30    |
```

Exemplo:

| Nome | Idade |
| ---- | ----- |
| Ana  | 25    |
| João | 30    |

---
